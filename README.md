## 🌱 Overview

**FarmSwift** is a smart agriculture marketplace inspired by quick-commerce platforms like Blinkit and Zepto.

It connects **Farmers**, **Customers**, **Delivery Partners**, and **Dark Store Partners** on a single platform to enable seamless ordering, product management, and delivery.

The entire backend is powered by **Google Apps Script** and **Google Sheets**, making it lightweight, serverless, and easy to deploy.
